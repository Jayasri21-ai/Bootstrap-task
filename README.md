**Introduction**

This project is a responsive multi-page website developed as part of an assignment to demonstrate practical knowledge of Bootstrap 5 and front-end web development. The objective of the project is to design clean, user-friendly web pages using reusable UI components, responsive layouts, and modern styling practices. The website consists of a Home page, About page, and Contact page, all built using Bootstrap’s grid system and utility classes to ensure consistency and responsiveness across devices. The project also includes version control and live deployment to showcase real-world development workflow.**Step-by-Step Project

Introduction

This project is a responsive multi-page website developed as part of an assignment to demonstrate practical knowledge of Bootstrap 5 and front-end web development. The objective of the project is to design clean, user-friendly web pages using reusable UI components, responsive layouts, and modern styling practices. The website consists of a Home page, About page, and Contact page, all built using Bootstrap’s grid system and utility classes to ensure consistency and responsiveness across devices. The project also includes version control and live deployment to showcase real-world development workflow.**Step-by-Step Project

**Process**

1.Requirement Analysis Understood the task requirements and planned a three-page responsive website using Bootstrap 5.

2.Project Setup Created the file structure (index.html, about.html, contact.html) and linked Bootstrap 5 via CDN.

3.Page Development

Home page with navbar, hero section, feature cards, and footer

About page with project details and service cards

Contact page with a responsive form

4.Styling & Responsiveness Used Bootstrap grid system and utility classes to maintain consistent layout and responsiveness across devices.

5.Testing & Review Verified navigation links, form inputs, and responsive behavior on different screen sizes.

6.Version Control & Deployment Pushed the project to GitHub, added a README file, and deployed the website using GitHub Pages / Netlify.

7.Reflection & Learning Documented the development process, tools used, challenges faced, learning outcomes, and time taken.
